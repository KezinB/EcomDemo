# 🔋 VoltKart Marketplace

A high-performance, immersive electronic components marketplace featuring a modern **Glassmorphism** design and a streamlined **Enquiry-based** checkout system.

![VoltKart Banner](https://images.unsplash.com/photo-1518770660439-4636190af475?auto=format&fit=crop&q=80&w=1200)

## ✨ Core Features

- 🛒 **Sidebar Enquiry Cart**: A smooth, slide-out drawer for managing component requests without leaving the catalog.
- 💎 **Premium Glass UI**: Modern aesthetic using advanced CSS backdrop filters and refined typography.
- ⚡ **Real-time Catalog**: Powered by Firebase Firestore for instantaneous updates and search filtering.
- 🛠️ **Admin Dashboard**: A comprehensive management panel for adding, editing, and monitoring inventory.
- 📱 **Fully Responsive**: Optimized for desktop, tablet, and mobile sourcing teams.

## 🛠️ Technology Stack

- **Frontend**: Vanilla HTML5, CSS3 (Modern Flex/Grid), JavaScript (ES6+ Mobile-First)
- **Database**: Cloud Firestore (Firebase)
- **Styling**: Custom CSS Design System with Glassmorphism blur effects

## 🚀 Getting Started

1. **Clone the Repo**:
   ```bash
   git clone https://github.com/KezinB/EcomDemo.git
   ```
2. **Setup Firebase**:
   Replace the Firebase configuration object in `script.js` with your own project credentials.
3. **Run Locally**:
   Open `index.html` using a local server (like Live Server in VS Code).

## 🔒 Security Note
This project uses **Firestore Security Rules** that should be restricted to authenticated administrators before moving to a production environment.
